# Aplikasi Penjualan Desktop (MVC)

## Deskripsi
Aplikasi Penjualan Desktop ini dikembangkan sebagai **Tugas Ujian Akhir Semester (UAS)**  
mata kuliah **Pemrograman Visual (Desktop)** pada Program Studi **Teknik Informatika**.  
Aplikasi ini merupakan aplikasi penjualan sederhana yang menerapkan **arsitektur MVC (Model–View–Controller)** untuk memisahkan logika bisnis, tampilan, dan pengolahan data.

---

## Tujuan Pengembangan
- Menerapkan konsep **Pemrograman Visual berbasis Desktop**
- Mengimplementasikan pola arsitektur **MVC**
- Membuat aplikasi penjualan yang terhubung dengan database
- Memenuhi tugas UAS Pemrograman Visual

---

## Fitur Aplikasi
- Login pengguna
- Pengelolaan data barang
- Pengelolaan data pelanggan
- Transaksi penjualan
- Penyimpanan dan pengolahan data ke database
- Laporan penjualan

---

## Arsitektur MVC
Aplikasi ini menggunakan pola **MVC (Model–View–Controller)**, yaitu:
- **Model**  
  Bertanggung jawab terhadap pengolahan data, koneksi database, serta proses CRUD.
- **View**  
  Berupa form-form antarmuka pengguna (Windows Form).
- **Controller**  
  Menghubungkan View dan Model serta mengatur alur logika aplikasi.

---

## Teknologi yang Digunakan
- Bahasa Pemrograman : **VB.NET**
- Platform            : **Windows Desktop (Windows Forms)**
- IDE                 : **Visual Studio**
- Database            : **MySQL**
- Version Control     : **Git & GitHub**
